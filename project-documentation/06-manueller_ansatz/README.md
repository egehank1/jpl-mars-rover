# Entwicklungsprozess – Status-LED & AutoFOCUS-orientierter Ansatz

Dieser Ordner dokumentiert die Entwicklung einer **Status-LED** für den Mars Rover
sowie den **manuellen, AutoFOCUS-orientierten Ansatz** von der Modellidee
über generator-ähnlichen Code bis zur ROS2-Integration.

## Inhalt
1. Ziel & Motivation  
2. Arbeitsablauf im Projekt  
3. ROS2 Status-LED  
4. AutoFOCUS-Modell (konzeptionell)  
5. Generator-ähnlicher Code  
6. ROS2 Wrapper / Adapter  
7. Build & Start  
8. Probleme & Lösungen  
9. Fazit & Ausblick
