# Manueller Ansatz: AutoFOCUS-orientierte Codeintegration in ROS2

Dieser Ordner dokumentiert den **manuellen Ansatz zur Integration AutoFOCUS-ähnlich generierten Codes in ein ROS2-System**.

Der Fokus liegt darauf zu zeigen, wie:

- ein **modellbasierter Ansatz** (AutoFOCUS)
- über **Codegenerierung**
- und einen **Wrapper/Adapter**
- sauber an ein bestehendes **ROS2-System**

angebunden werden kann.

Die Dokumentation ist bewusst **code-zentriert** aufgebaut und enthält **einige relevanten Dateien**.
